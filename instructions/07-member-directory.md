# 成員目錄

## 目標
實作可搜尋的成員目錄，支援按身份和屆別篩選，讓學弟妹能找到校友聯絡方式。

## 前置條件
- 06-profile.md 已完成

## 執行步驟

### 1. 建立成員目錄頁 src/screens/MemberDirectoryScreen.tsx
此頁面位於「探索」Tab 下方的子頁面。

頂部搜尋欄：即時搜尋顯示名稱。

篩選選項：
- 身份：全部、現役社員、校友、一般用戶
- 校友屆別：下拉選單顯示所有有校友的年份
- 帳號等級：Lv1 到 Lv5

列表顯示每位成員：
- 頭像、顯示名稱、身份標籤
- 帳號等級徽章
- Bio 前 50 字
- 若為校友且有填 LinkedIn 或 Instagram，顯示對應小圖示

點擊任一成員導向其公開頁面。

### 2. 分區顯示
校友和現役社員分開顯示但在同一頁面：
- 頂部顯示現役社員區塊（標題：現役社員）
- 下方顯示校友區塊（標題：ROCSAUT 校友）
- 套用篩選後兩個區塊都同步過濾

### 3. 實作無限滾動
呼叫 GET /api/community/users，支援分頁載入。
滾動到底部自動載入下一頁。

## 驗證方法
1. 確認成員列表正確載入
2. 確認搜尋功能即時過濾結果
3. 確認身份篩選正確分類
4. 確認校友屆別篩選正確過濾
5. 確認點擊成員能進入其公開頁面
完成後 commit 並 push 至 flyincloud2001/ROCSAUT-Community。
