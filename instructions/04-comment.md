# 留言功能

## 目標
實作留言、巢狀回覆、匿名留言、最佳回覆標記功能。

## 前置條件
- 03-post.md 已完成

## 執行步驟

### 1. 建立留言元件 src/components/CommentItem.tsx
- 作者頭像、顯示名稱、身份標籤、時間
- 若匿名則顯示「匿名用戶」
- 留言內容
- 按讚/倒讚按鈕和數字
- 回覆按鈕（點擊後在輸入欄顯示「回覆 @用戶名」）
- 若為最佳回覆顯示金色「最佳回覆」標籤
- 作者和管理員可看到刪除按鈕

### 2. 實作巢狀回覆
最多支援兩層巢狀（留言和留言的回覆）。
第二層回覆縮排顯示，有左側色條標示。
呼叫 POST /api/community/posts/:id/comments 時帶入 parentId。

### 3. 建立留言輸入區 src/components/CommentInput.tsx
固定在發文詳細頁底部。
包含：文字輸入欄、匿名開關（需 Lv2 以上）、送出按鈕。
點擊回覆某則留言時，輸入欄頂部顯示「回覆 @用戶名」並可取消。

### 4. 實作最佳回覆標記
只有發文者可以標記最佳回覆。
每篇發文只能有一個最佳回覆。
點擊後呼叫 POST /api/community/posts/:id/comments/:commentId/best。
最佳回覆置頂顯示在留言列表最上方。

### 5. 實作刪除留言
作者和管理員都可以刪除。
刪除前顯示確認對話框。
若刪除的留言有子回覆，顯示「此留言已刪除」而非直接消失。

## 驗證方法
1. 確認留言列表正確顯示
2. 確認新增留言後即時出現
3. 確認巢狀回覆正確縮排
4. 確認匿名留言顯示為「匿名用戶」
5. 確認最佳回覆置頂並顯示金色標籤
6. 確認刪除留言後正確處理子回覆
完成後 commit 並 push 至 flyincloud2001/ROCSAUT-Community。
