# 發文功能

## 目標
實作建立發文、查看發文詳細頁、編輯和刪除發文功能。

## 前置條件
- 02-feed.md 已完成

## 執行步驟

### 1. 建立發文頁面 src/screens/CreatePostScreen.tsx
- 標題輸入欄（必填）
- 內文輸入欄（必填，多行）
- 分類選擇下拉選單
- 圖片附件按鈕（依帳號等級限制數量）
- 匿名發文開關（需 Lv2 以上）
- 右上角發布按鈕
- 發布前驗證：標題和內文不得為空
- 每日發文數量限制（依帳號等級）

### 2. 實作圖片上傳
使用 expo-image-picker 選取圖片。
上傳至 Supabase Storage（bucket：community-images）。
上傳完成後顯示縮圖預覽，可點擊刪除。
Lv1 不顯示圖片附件按鈕。
Lv2 最多 1 張，Lv3 以上最多 3 張。

### 3. 建立發文詳細頁 src/screens/PostDetailScreen.tsx
- 頂部：作者資訊（同發文卡片規格）
- 完整標題和內文
- 圖片輪播（若有）
- 按讚/倒讚按鈕和數字
- 留言數
- 發文者可看到「標記最佳回覆」選項
- 作者和管理員可看到「編輯」和「刪除」按鈕
- 下方顯示留言列表（見 04-comment.md）

### 4. 實作編輯發文
只有作者可以編輯自己的發文。
編輯頁面和建立頁面共用同一個元件，預填現有內容。
呼叫 PATCH /api/community/posts/:id。

### 5. 實作刪除發文
作者和管理員都可以刪除。
刪除前顯示確認對話框。
呼叫 DELETE /api/community/posts/:id。
刪除後返回動態牆並重新整理。

## 驗證方法
1. 確認建立發文後出現在動態牆
2. 確認 Lv1 用戶無法附圖且無法匿名
3. 確認圖片上傳後正確顯示縮圖
4. 確認發文詳細頁正確顯示所有資訊
5. 確認編輯後內容正確更新
6. 確認刪除後從動態牆消失
完成後 commit 並 push 至 flyincloud2001/ROCSAUT-Community。
