# 分類標籤系統

## 目標
建立分類標籤系統，讓發文可以被分類，方便用戶瀏覽和篩選。

## 前置條件
- 09-search.md 已完成

## 執行步驟

### 1. 預設分類
在資料庫中建立以下預設分類（由 Claude Code 直接在 Supabase 執行 SQL 建立）：
- 問答：學弟妹提問、經驗分享
- 求職：工作機會、面試經驗、實習資訊
- 生活：多倫多生活、住宿、交通
- 課業：選課建議、教授評價、讀書資源
- 活動：社團活動討論、揪團
- 閒聊：隨意聊天、趣事分享
- 公告：官方公告（只有管理員可發文至此分類）

### 2. 建立分類選擇元件 src/components/CategoryPicker.tsx
建立發文時顯示的分類選擇器。
水平滾動的分類標籤列表，點擊選取。
已選取的分類顯示主色紅背景。

### 3. 建立分類標籤元件 src/components/CategoryBadge.tsx
在發文卡片底部顯示分類名稱的小標籤。
點擊後導向該分類的瀏覽頁面。

## 驗證方法
1. 確認預設分類正確建立在資料庫
2. 確認建立發文時可以選擇分類
3. 確認發文卡片正確顯示分類標籤
4. 確認點擊分類標籤能進入分類瀏覽頁
完成後 commit 並 push 至 flyincloud2001/ROCSAUT-Community。
